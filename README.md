# Kanaa Kanden: An Andal Kalyanam

> **Rahul Babu M V weds Suruthi R**  
> *September 13, 2026 · Thaibala Wedding Hall, Tenkasi*

---

A cinematic, single-page wedding invitation website built with Vanilla HTML, CSS, and JavaScript — themed around Andal's *Nachiyar Thirumozhi* and the divine romance of the Andal–Krishna union.

## ✨ Features

- **Animated Hero** — Gold shimmer title with Tamil verse, Great Vibes script names, gopuram SVG accent
- **Countdown Timer** — Real-time countdown to Muhurtham (Sept 13, 2026 · 8:30 AM IST)
- **Andal Verse** — Tamil text, IAST transliteration, and English translation
- **Event Cards** — Engagement, Reception, and Muhurtham with hover animations
- **Venue + Maps** — Google Maps navigation button for Thaibala Wedding Hall
- **Audio Player** — Nadaswaram background music with play/pause and volume
- **R&S Monogram** — Circular lotus mandala SVG logo in Temple Antique Gold
- **Scroll Animations** — IntersectionObserver fade-up transitions
- **Responsive** — Adapts elegantly to desktop, tablet, and mobile

## 🚀 Getting Started

```bash
npm install
npm run dev      # Starts at http://localhost:5179
```

### Adding Music

Place `nadaswaram.mp3` (Sita Kalyana Vaibhogame) in the project root. The audio player will detect it automatically.

## 🌐 Live Preview

Open `index.html` directly in any modern browser — no build step required.
